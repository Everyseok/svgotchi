# Stage 5: Hugging Face Ultra-Light Model Review

## Goal
Research local ultra-light models for TransitionPlan intent and emotion planning.

## Required Work
- Search Hugging Face candidates.
- Check model architecture, size, license, English-first intent/emotion fit, optional multilingual notes, browser runtime compatibility, offline vendoring feasibility, model card quality, intended use, limitations, and risks.
- Compare candidates in docs/model-review.md.
- Do not implement model runtime.

## Stop Condition
Stop and request user approval for the selected model candidate or the conclusion that no acceptable model exists.
