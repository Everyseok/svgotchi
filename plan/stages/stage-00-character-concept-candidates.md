# Stage 0: Character Concept Candidates

## Goal
Define the visual identity before code.

## Required Work
- Create 3 to 5 cute pixel Tamagotchi character concepts.
- Keep each concept compatible with a 100x100 SVG layout.
- Keep each concept riggable with simple body, eyes, mouth, blush, and optional simple appendages.
- Avoid complex unriggable artwork.
- Present pros, cons, and one recommendation.

## Stop Condition
Stop and request user approval for one concept before Stage 1.
