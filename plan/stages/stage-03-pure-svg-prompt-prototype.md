# Stage 3: Pure SVG Prompt Prototype

## Goal
Implement the prompt area using SVG elements only.

## Required Work
- Implement rect/text/caret/send-zone prompt UI.
- Implement focus handling on the SVG root.
- Implement keydown handling.
- Implement IME composition handling.
- Implement Enter submit and Backspace by Unicode code point.
- Implement caret blink and safe text clipping/truncation.
- Do not use foreignObject, HTML input, HTML button, contenteditable, or an HTML form.

## Stop Condition
Verify prompt behavior in target SVG document context and report results.
